solr:run

Full name:

io.github.borisnaguet:solr-maven-plugin:0.5.0:run

Description:

Starts Solrcloud and blocks. This is to be used for command-line only This is equivalent to "start-solrcloud -Dsolr.keep.running"

Attributes:

  • Requires a Maven project to be executed.
