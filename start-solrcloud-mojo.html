solr:start-solrcloud

Full name:

io.github.borisnaguet:solr-maven-plugin:0.2.0:start-solrcloud

Description:

Starts a new SolrCloud instance Can only be called once

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: pre-integration-test.
